# Panacare Administrator

This is a Next.js application for managing the Panacare platform. It provides a dashboard for administrators to manage various aspects of the system.

## Features

The Panacare administrator dashboard includes the following features:

- **Dashboard**: A central hub that provides a comprehensive overview of the platform's key metrics. It displays statistics and visualizations for patients, doctors, hospitals, and subscriptions, allowing administrators to monitor the platform's health and performance at a glance.

- **Patients**: This feature allows administrators to manage all patient-related information. Key functionalities include:
  - Viewing a list of all registered patients.
  - Accessing detailed patient profiles.
  - Monitoring patient activity, including active and inactive status.

- **Doctors**: Administrators can manage doctors on the platform through this feature. It includes:
  - A complete list of all doctors.
  - Detailed profiles for each doctor, including their verification status.
  - Tracking of doctor activity and engagement.

- **Hospitals**: This section enables the management of healthcare facilities associated with Panacare. Administrators can:
  - View a list of all registered hospitals.
  - Access detailed information about each facility.
  - Monitor the operational status of hospitals.

- **Appointments**: This feature provides tools for managing patient appointments. It allows administrators to:
  - View and track all scheduled appointments.
  - Monitor appointment statuses and details.

- **Teleconsultation**: The teleconsultation feature allows for the management of remote consultations between patients and doctors. Administrators can:
  - Oversee all teleconsultation sessions.
  - Track the status and details of each consultation.

- **Follow-Up Compliance Monitor**: This feature helps in tracking patient adherence to follow-up appointments and treatment plans. It provides:
  - A monitoring system for follow-up compliance.
  - Detailed reports on patient follow-up activities.

- **Subscriptions**: Administrators can manage the various subscription packages offered by Panacare. This includes:
  - Viewing a list of all available subscription packages.
  - Tracking the number of subscribers for each package.

- **Package Payment Tracker**: This feature provides a system for tracking payments related to subscription packages. It allows administrators to:
  - Monitor the payment status of all subscriptions.
  - Generate reports on package payments.

- **Risk Segmentation**: This feature is designed to categorize patients based on their health risks. It helps in:
  - Identifying high-risk patients for proactive care.
  - Implementing targeted health management strategies.

- **Blog Management**: This feature allows administrators to manage the content of the Panacare blog. It includes:
  - Creating, editing, and publishing blog posts.
  - Managing blog categories and tags.

- **Audit Logs**: The audit logs provide a record of all activities performed within the administrator dashboard. This is crucial for:
  - Security and compliance monitoring.
  - Tracking changes and actions taken by administrators.

## Tech Stack

- **Next.js**: A React framework for building server-side rendered and static web applications.
- **React**: A JavaScript library for building user interfaces.
- **Tailwind CSS**: A utility-first CSS framework for rapid UI development.
- **Chart.js**: A library for creating responsive and interactive charts.
- **recharts**: A composable charting library built on React components.
- **lucide-react**: A library of simply designed icons for React.
- **react-icons**: A collection of popular icon libraries for React.
- **jspdf** & **jspdf-autotable**: Libraries for generating PDF reports.
- **papaparse**: A library for parsing CSV files.
